# House Price Prediction using Machine Learning

## Overview
This project is my first hands-on Machine Learning project.
The goal is to understand ML concepts through implementation rather than theory.

Given house features like area, number of bedrooms, and location,
the model predicts the house price.

---

## Problem Statement
Predict the price of a house based on historical data.

- Input: area, bedrooms, bathrooms, location
- Output: house price (regression problem)
